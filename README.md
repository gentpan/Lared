# Lared

一款为写作者打造的极简 WordPress 主题。

- 作者: [西风](https://xifeng.net)
- 主题主页: [xifeng.net/wordpress-lared-theme.html](https://xifeng.net/wordpress-lared-theme.html)

## 快速开始

1. 将主题目录放到 `wp-content/themes/Lared`
2. 后台启用主题
3. 打开 `外观 -> Lared 设置` 完成基础配置
