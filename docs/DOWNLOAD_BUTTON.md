# 下载按钮短代码使用说明

> Lared 主题内置下载按钮短代码，方便在文章中添加美观的下载链接。

---

## 基础用法

```
[download_button dl_url="https://example.com/file.zip" dl_name="文件名.zip"]
```

---

## 完整参数

| 参数 | 说明 | 必需 | 默认值 |
|------|------|------|--------|
| `dl_url` | 下载链接 | ✅ | - |
| `dl_name` | 文件名称 | ❌ | "未知文件" |
| `dl_text` | 按钮文字 | ❌ | "立即下载" |
| `dl_size` | 文件大小 | ❌ | - |
| `dl_format` | 文件格式 | ❌ | - |
| `dl_version` | 版本号 | ❌ | - |
| `dl_note` | 备注说明 | ❌ | - |

---

## 使用示例

### 示例 1：基础用法

```
[download_button 
    dl_url="https://dl.xifengcdn.com/2025/08/heicconverted.rar" 
    dl_name="HEIC转换脚本"
]
```

### 示例 2：完整参数

```
[download_button 
    dl_url="https://dl.xifengcdn.com/2025/08/heicconverted.rar" 
    dl_name="HEIC转换脚本" 
    dl_text="本站下载" 
    dl_size="1.4 KB" 
    dl_format="RAR" 
    dl_version="v1.0.0" 
    dl_note="解压即用"
]
```

## 样式特点

- 📦 **文件信息标签**：格式、版本、大小分别用不同颜色标识
- 💡 **备注提示**：黄色提示条显示重要信息
- 🎯 **醒目按钮**：红色主题色下载按钮
- 📱 **响应式设计**：移动端自动适配

## 常见问题

### Q: 下载链接不能为空？

**A:** 如果 `dl_url` 为空，短代码会显示错误提示：
> 错误：请提供下载链接

### Q: 可以使用外部链接吗？

**A:** 可以，支持任何有效的 URL，包括：
- 本站资源 `/wp-content/uploads/file.zip`
- 外部 CDN `https://cdn.example.com/file.zip`
- 网盘链接等

### Q: 如何在编辑器中使用？

**A:** 
1. 在文章编辑器中直接粘贴短代码
2. 或在 "自定义 HTML" 区块中使用
