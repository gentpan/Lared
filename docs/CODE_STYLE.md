# 内联代码样式说明

> Lared 主题支持多种内联代码样式，包括普通代码和红色高亮代码。

---

## 样式类型

### 1. 普通内联代码

默认样式：**淡红背景 + 深红字**

```markdown
这是 `.heic` 格式的文件
```

效果：
- 背景：淡红色（8%透明度）
- 文字：深红色（#c41e3a）

---

### 2. 红色高亮代码

用于特殊内容如 **UUID、密钥、验证码** 等

#### 方式 1：使用 class

```html
<code class="code-red">123e4567-e89b-12d3-a456-426614174000</code>
或
<code class="red">123e4567-e89b-12d3-a456-426614174000</code>
```

#### 方式 2：使用短代码（推荐）

```
[red_code]123e4567-e89b-12d3-a456-426614174000[/red_code]
```

或

```
[red_code text="123e4567-e89b-12d3-a456-426614174000"]
```

效果：
- 🔴 **红底白字**
- 加粗显示
- 字符间距加大（易于阅读）

---

## 效果对比

| 类型 | 样式 | 适用场景 |
|------|------|----------|
| 普通代码 | 淡红底 + 深红字 | 一般代码、文件名、路径 |
| 红色高亮 | 🔴 红底白字 | UUID、密钥、验证码、重要标识 |

---

## 使用示例

### 示例 1：普通代码

```markdown
文件路径是 `/wp-content/themes/`，格式为 `.heic`
```

显示：
- `/wp-content/themes/` - 淡红底深红字
- `.heic` - 淡红底深红字

### 示例 2：红色高亮代码

```markdown
您的激活码是 [red_code]ABC-123-XYZ[/red_code]
```

显示：
- `ABC-123-XYZ` - 🔴 红底白字（醒目）

### 示例 3：UUID

```markdown
设备 UUID: [red_code]123e4567-e89b-12d3-a456-426614174000[/red_code]
```

或使用 HTML：

```html
设备 UUID: <code class="code-red">123e4567-e89b-12d3-a456-426614174000</code>
```

## 技术实现

### 普通代码样式

```css
code {
    color: #c41e3a;
    background: rgba(245, 48, 4, 0.08);
}
```

### 红色高亮代码样式

```css
code.code-red {
    color: #fff;
    background: #f53004;
    font-weight: 600;
    letter-spacing: 0.5px;
}
```
